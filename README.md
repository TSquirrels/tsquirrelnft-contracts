# TSquirrel NFT Contracts
A repo of TSquirrel NFT ethereum contracts.

| Contract | Description |
| ----------- | ----------- |
| TSquirrelNFT | Enumerable ERC721 NFT contract with buyable mints at a fixed native EVM currency price. |

## Build

```
truffle build
```

## Migrate

```
truffle migrate
```

## Test

```
truffle test
```
